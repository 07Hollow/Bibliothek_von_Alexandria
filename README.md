This repository contains a comprehensive knowledge base on Physics, Computer Science, and various other subjects. Created with **Obsidian**, a powerful knowledge management tool, and exported using the **Perlite** plugin, this project aims to provide an easily accessible and navigable resource for learning and reference.

# Features
**Extensive Content:** Covers a wide range of topics in Physics, Computer Science, and more.
**Obsidian Integration:** Utilizes Obsidian for note-taking and organization.
**Static Site Generation:** Uses the Perlite plugin to generate static HTML files from Obsidian notes.
**Netlify Deployment:** Deployed on Netlify for seamless access and hosting.

# Usage
Browse the Content: Navigate through the various topics and notes for detailed information and learning resources.
Update and Contribute: Contributions are welcome! Feel free to add new notes or update existing ones to enhance the knowledge base.

# Deployment
The static site generated by Perlite is hosted on **Netlify**, ensuring fast and reliable access. Any updates pushed to this repository are automatically deployed to keep the site up-to-date.
